# Movie Finder - JSPM + ES6

A Movie Search App made with JSPM + ES6 and themoviedb.org API

## Installation

Download or clone the project:

```sh
npm install
npm run jspm install
npm run dev
```

## Usage

In the project directory, you can run:

### `npm run dev`

### `npm run prod`
